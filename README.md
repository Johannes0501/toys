# toys

Deployed via [theArchitect](https://github.com/your-org/theArchitect)

## Deployment

This project is automatically deployed when you push to the `main` branch.

- **URL**: https://toys.xn--datastrm-c5a.com
- **Health Check**: https://toys.xn--datastrm-c5a.com/api/health

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Configuration

Project configuration is stored in `project.yaml`.
